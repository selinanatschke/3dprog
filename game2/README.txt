Instructions to execute game: 

1. open godot.exe (version 3.5.2)
2. click import 
3. navigate to the game folder and double click project.godot
4. click import & edit
5. press F6 (or the play button in the top right corner)
=> a window should open 

additional note: every time you open the project, it changes something so git notices. to "revert" these changes, go to your command line and use the command "git reset --hard" - it will throw away uncommited changes
